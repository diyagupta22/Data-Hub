# Prompts Used – Data Hub REST API Project

1. How to create a basic Express server in Node.js.

2. How to build REST API routes for CRUD operations (GET, POST, PUT, DELETE).

3. How to store and manage data using a mock array as a database.

4. How to test API endpoints using Postman.

5. How to implement custom middleware to log request method, URL and time.

6. How to create a simple login route that returns a fake JWT token.

7. How to handle errors when updating or deleting a post that does not exist.
